1. Даны два целых числа A и B (A < B). Найти сумму всех целых чисел от A до B включительно.

2.  Даны два целых числа A и B (A < B). Найти произведение всех целых чисел от A до B включительно.

3. Даны два целых числа A и B (A < B). Найти сумму квадратов всех целых чисел от A до B включительно.

4. Дано вещественное число A и целое число N (> 0). Используя один цикл, найти значение 
выражения 1 – A + A2 – A3 + … + (–1)N·AN. Условный оператор не использовать.

5.  Дано целое число N (> 0). Найти произведение N! = 1·2·…·N (N–факториал). 
Чтобы избежать целочисленного переполнения, вычислять это произведение с помощью вещественной переменной 
и вывести его как вещественное число.

6. Дано целое число N (> 1). Последовательность чисел Фибоначчи FK (целого типа) 
определяется следующим образом: F1 = 1, F2 = 1, FK = FK–2 + FK–1, K = 3, 4, … . Вывести элементы F1, F2, ..., FN.
